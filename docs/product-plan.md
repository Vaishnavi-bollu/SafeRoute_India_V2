# SafeRoute India Product Plan

## Problem statement
People in crowded events and public places struggle to identify the safest route in real time. SafeRoute India solves this by recommending routes based on crowd density, lighting quality, and live danger reports.

## Core experience
- Secure login and profile setup
- Live route recommendation with a dynamic safety score
- Visual heatmap for crowd and lighting conditions
- SOS emergency flow for urgent help
- Nearby safe places and emergency shelters

## MVP scope
- Splash + login
- Home dashboard with recommended route
- Live map and heatmap screens
- Safety explanation screen and SOS flow
- Nearby safe places and settings

## Hackathon pitch angle
A beautiful, real-time safety navigation app that feels premium, practical, and immediately useful in high-density urban environments.
