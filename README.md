# SafeRoute India

SafeRoute India is an AI-powered safety navigation concept for crowded events and public places. This workspace contains a premium mobile-first React experience for the product, plus a complete hackathon architecture and execution plan.

## What is included
- Premium glassmorphism UI with dark mode and safety color accents
- Screens for splash, login, home, live map, heatmap, route details, SOS, nearby safe places, and settings
- Architecture, data schema, API plan, AI workflow, and development roadmap

## Run locally
```bash
npm install
npm run dev
```

## Project vision
SafeRoute India recommends the safest route instead of only the shortest route by combining crowd density, lighting conditions, live danger reports, and time-based safety scoring.
